# Chrome-palette

A SublimeText/Atom/Vscode style command palette for chrome!

![alt text](screenshot.png)

## Features

- [x] Commands sorted by usage
- [x] Fuzzy search of commands
- [x] Search open tabs
- [x] Search history
- [x] Search in gdrive

## Development

```bash
npm install
npm run start
```

## Manual testing

Import the dist folder as an unpacked extension in chrome.
